Calculator using wxFormBuilder¶

The Project is to create a Gui for Calculator. Python code for GUI is generated with WxFormBuilder. One need to open .fbp (e.g.calc.fsb for our code) and modify the GUI. Then click generate, to generate corresponding python code (gui.py) Different functions used in put OnButtonClick for GUI is defined here.
